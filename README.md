# Sistem Informasi SPJ DPRD

Website Sistem Informasi SPJ DPRD berbasis PHP Native dan MySQL.

## Fitur
- Login Admin dan User
- Dashboard Admin
- Dashboard User
- Tambah SPJ
- Edit SPJ
- Hapus SPJ
- Data SPJ
- Data User
- Pencarian SPJ

## Teknologi
- HTML
- CSS
- PHP Native
- MySQL

## Cara Menjalankan
1. Clone repository
2. Import database `spj_dprd.sql`
3. Letakkan folder di `htdocs` (XAMPP) atau `www` (Laragon)
4. Jalankan Apache dan MySQL
5. Buka:
```
http://localhost/spj_dprd
```
